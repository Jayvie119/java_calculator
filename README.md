# Java Calculator

## Overview
This Java Calculator project is a console-based application that performs basic arithmetic operations including addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Jayvie119/java_calculator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd java_calculator
   ```
3. Compile the Java files:
   ```bash
   javac Calculator.java
   ```
4. Run the application:
   ```bash
   java Calculator
   ```

## Usage
Follow the prompts in the console to perform calculations.
